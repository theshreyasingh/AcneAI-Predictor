# AcneAI – Lifestyle-Based Acne Risk Prediction System

## Overview

AcneAI is a Machine Learning project that predicts acne severity based on lifestyle factors such as:

- Sleep Hours
- Water Intake
- Stress Level
- Oily Food Consumption

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Streamlit

## Features

- Acne severity prediction
- Interactive web interface
- Lifestyle-based recommendations
- Feature importance analysis

## How to Run

```bash
python -m streamlit run app.py
